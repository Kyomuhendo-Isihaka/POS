Project Name: Lorry & Truck Services Management System

Developed By: GPW Team

Summary: It is a simple mini project which was created to facilitate easy management of Vehicles & Cargo services of a company. Here Admin has all privileges and can register new cargo shipments, add new vehicles, add new companies, update cargo information, update payments and track shippings thru this sytem.
 
Technologies Used: HTML, CSS for frontend, PHP as a backend and MySQL for the database.

Features:
-->Simple and elegant UI
-->Easy to use
-->Easy to add and maintain all vehicle & its cargo information
-->Payments balance tracking
-->Contains all essential functionalities
-->Fully customizable
-->Compact in size
